Setup in 5 minutes

Share this with your classmates — everyone can have their own auto-updating calendar.

1

Fork this repository

Click Fork on GitHub. This creates your own copy of the project.

2

Edit calendar-config.json

Replace the modules with yours from your contrat d'études. Set your TD group number for each module.

{
  "startDate": "2026-01-19",
  "endDate": "2026-08-31",
  "modules": [
    { "code": "MIN17212", "name": "Simulation", "tdGroup": 1 },
    { "code": "MSANGS2I", "name": "Anglais", "tdGroup": 4 }
  ]
}
3

Enable GitHub Pages

Go to Settings → Pages → Source: select main branch, / (root) folder, click Save.

4

Enable Actions

Go to Settings → Actions → General, select "Allow all actions", click Save.

5

Run the workflow

Go to Actions → Update Calendar → Run workflow. Wait for the green ✓ (~30 seconds).

6

Subscribe!

Add https://<your-username>.github.io/Fetcher/calendar.ics to your calendar app. Done!

How to subscribe

📱

Google Calendar

Other calendars (+) → From URL → paste your calendar link.

🍎

iPhone / iPad

Settings → Calendar → Accounts → Add → Other → Subscribed → paste the link.

📧

Outlook

Add calendar → Subscribe from web → paste the link.

🤖

Android

Open the URL in your browser — it will offer to add events to your calendar.
